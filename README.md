# Online-Voting-System
Let me just break down all the context in simple paragraph....

This lightweight, console-based Python application allows voters to securely register, log in, and cast a single vote for their preferred candidate while managing all records locally inside a structured JSON file. The system's core context features an authentication module for verifying unique user credentials, a voting panel that strictly enforces a one-vote-per-user limit, and a protected administrative dashboard (accessible via `admin` / `1234`) that aggregates and displays real-time election results.

Here Admin and Voters can be edited on your choice of votes..
